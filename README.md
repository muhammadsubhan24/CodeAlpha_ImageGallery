# CodeAlpha - Image Gallery

A beautiful, responsive image gallery with modern hover effects and a smooth lightbox modal viewer. Built as part of the CodeAlpha Web Development Internship.

## Features
- **Responsive Grid Layout**: Auto-adjusts seamlessly across mobile, tablet, and desktop screens.
- **Smooth Hover Zoom**: Interactive zoom and dark overlay title effects on image hover.
- **Interactive Lightbox**: Click on any image to open it in a high-quality popup modal view.
- **Navigation Controls**: Smooth transition buttons (Next/Previous) to navigate through images in lightbox view.

## Tech Stack
- HTML5
- CSS3 (Grid, Flexbox & CSS Keyframes)
- JavaScript (ES6+)
